# 研发需求推进管理工具 Windows 桌面版

## 启动

双击 `Start-RD-Tracker.cmd`。

启动器会在本机 `127.0.0.1` 临时启动一个本地服务，然后用 Microsoft Edge 的应用窗口打开工具。关闭应用窗口后，本地服务会自动退出。

## 数据位置

业务数据保存在本目录下的 `profile` 用户数据目录中，属于这个桌面应用窗口，不会混入普通浏览器配置。

工具内仍然支持：

- JSON 完整导出
- JSON 导入恢复
- CSV 需求列表导出
- Markdown 周报

## 文件结构

- `app/`：应用页面文件
- `profile/`：首次启动后生成，保存本地浏览器数据
- `logs/`：首次启动后生成，保存本地服务日志
- `launch.ps1`：桌面启动逻辑
- `Start-RD-Tracker.cmd`：双击入口

## 说明

这是本地优先桌面封装，不会上传业务数据。后续如果要做安装包级应用，可以在这个原型基础上迁移到 Tauri + SQLite。
